## Overview 🌟

The SmartLinx app allows users to control and monitor their smart devices effortlessly. The app supports a wide range of devices, providing a unified interface for managing all your smart home gadgets.

## Features ✨

- **Cross-Platform**: Runs on both Android and iOS devices 📱.
- **Device Management**: Add, remove, and control smart devices from a single app 💡.
- **Real-Time Monitoring**: View the status of your devices in real time 🕒.
- **Scheduling**: Schedule actions for your devices to automate daily routines ⏰.

## Requirements 🛠️

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK: Included with Flutter
- Android Studio or Xcode: For building and testing on Android and iOS respectively

## Installation 📥

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/SmartLinx-Project/Mobile-App.git
    cd mobile-app-master
    ```

2. **Install Dependencies**:
    ```bash
    flutter pub get
    ```

3. **Set Up for Android**:
    - Open the project in Android Studio.
    - Ensure you have an Android device or emulator running.

4. **Set Up for iOS**:
    - Open the project in Xcode.
    - Ensure you have an iOS device or simulator running.

## Usage 🚀

1. **Run the App**:
    ```bash
    flutter run
    ```

2. **Building for Release**:
    - Android:
      ```bash
      flutter build apk
      ```
    - iOS:
      ```bash
      flutter build ios
      ```

3. **Navigating the App**:
   - **Home Screen**: Displays the list of added devices 🏠.
   - **Device Control**: Tap on a device to view and control its settings 🔧.
   - **Scheduling**: Set schedules for your devices to automate tasks ⏰.
